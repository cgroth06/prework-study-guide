HTML

  • Head elements contain information about the webpage, but will not be visible to the user.
  • The title will however be displayed in the tab's text bar.
  • The body element contains the visible content show to the user.
  • head element is only visible in the tab name of the window
  • body element contains the visible child elements of the page
  • h1 through h6 can be used for decreasing header size
  • p is used to establish a paragraph
  • ul = unordered list, ol = unordered list, li = list item
  • img contains source src=, name or description alt=, width= pixels in quotes, height= pixels in quotes
  • br = empty line or break

Git

  • git status : checks what branch we are on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git flow w/ PR > see where you are > git status, switch to main > git checkout main, pull remote main to local main > git pull origin main, create a new branch > git checkout -b "branch-name" ex: feature/add-js, make your changes in vscode stage / add changes > git add -A commit changes with a message > git commit -m "add a message here" update changes to local branch > git pull origin main push changes to remote repository git push origin "branch-name" from above ex: feature/add-js go to Github to complete PR
